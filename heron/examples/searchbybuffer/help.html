About this Example

Test for search by buffer and vector layers

Usage

Search by vector layers

1)Draw something
2)Select "Editor" from vector layers combo
3)Select buffer distance in meters (see notes)
4)Press "buffer"
5)Select WFS layer to search (USA States is better)
6)Press "search"

Search by WFS layers

1)Select layer from WFS layers combo
2)Select buffer distance in meters (see notes)
3)Select features by drawing
4)Select WFS layer to search
5)Press "search"
